Note Manager with Tags — , мощные заметки с тегами
Коллекция из семи реализаций менеджера заметок с поддержкой тегов. Каждая версия включает:

Создание, редактирование, удаление заметок

Добавление тегов (один или несколько, автодополнение)

Фильтрация по тегам (клик по тегу показывает все заметки с ним)

Полнотекстовый поиск по заголовку и содержимому

Цветовая маркировка заметок (по тегам или вручную)

Автосохранение (каждые 30 секунд или при изменении)

Экспорт/импорт в JSON или Markdown

Тёмная/светлая тема (в GUI/веб-версиях)

Статистика (количество заметок, популярные теги)

Языки и технологии
Язык	Интерфейс	Хранение	Особенности
Python	Tkinter GUI	JSON	Автодополнение тегов, цветовая подсветка, drag&drop для тегов
JavaScript	Веб (HTML/CSS)	IndexedDB / localStorage	Real-time поиск, облако тегов, адаптивный дизайн
Go	Веб (HTML/template)	JSON	Быстрый поиск, REST API, поддержка Markdown
Rust	CLI + диалоги	JSON (serde)	Цветной вывод, интерактивное управление, экспорт в Markdown
Java	Swing GUI	JSON (Jackson)	JList с тегами, встроенный редактор, иконки
C#	Windows Forms	JSON / SQLite	DataGridView, контекстное меню, экспорт в HTML
PHP	Веб + MySQL/JSON	MySQL / JSON	Поддержка множественных тегов, облако, пагинация
